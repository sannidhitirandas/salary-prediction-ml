# 💼 Salary Prediction Model using Machine Learning

## 📌 Overview

This project predicts employee salaries based on features such as age, education, job title, and years of experience using Linear Regression.

---

## 🚀 Features

* Data cleaning and preprocessing
* Handling categorical variables using one-hot encoding
* Model training using Linear Regression
* Evaluation using MSE and R² score
* Data visualization with Matplotlib

---

## 📊 Model Performance

* R² Score: ~0.81
* Mean Squared Error: ~499M
* RMSE: ~22K (approx)

---

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Matplotlib

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python salary.py
```

---

## 📁 Dataset

Dataset used from Kaggle:
https://www.kaggle.com/datasets/mohithsairamreddy/salary-data

---

## 📢 Conclusion

This project demonstrates how machine learning can be used to predict salaries based on multiple features. It helped in understanding regression models, feature encoding, and evaluation metrics.
